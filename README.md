# GIDRA Barbershop — Система бронирования и управления

Современная платформа для онлайн-записи, администрирования и аналитики, разработанная специально для барбершопа **GIDRA** (г. Курган). Проект построен на базе **Laravel 11**, **Tailwind CSS** и **Blade**. Он предоставляет удобный интерфейс для записи клиентов и мощную панель управления для администраторов.

---

## 🌟 Ключевые возможности

### 📅 Система бронирования для клиентов
* **Динамический выбор времени:** Доступные слоты (время) генерируются на лету с учетом рабочего графика мастера, уже существующих записей и общей длительности выбранных услуг.
* **Защита от накладок (оверлапов):** Проверка на бэкенде гарантирует, что два клиента не смогут записаться к одному мастеру на одно и то же время.
* **Зависимость цены от ранга мастера:** Стоимость услуг автоматически пересчитывается в зависимости от категории выбранного барбера (`Junior Barber`, `Barber` или `Top Barber`).
* **Защита от спама:** Форма бронирования защищена капчей Google reCAPTCHA.

### 💬 Автоматизированные отзывы по токенам
* **Генерация токенов:** Когда администратор переводит статус записи в «Выполнено» (`completed`), система автоматически создает уникальный токен и отправляет клиенту на email ссылку для написания отзыва.
* **Модерация отзывов:** Новые отзывы не публикуются сразу — они попадают в очередь на модерацию в админ-панели.
* **Плавный слайдер:** Одобренные отзывы выводятся на главной странице в виде бесконечной карусели на чистом JavaScript, которая плавно останавливается при наведении курсора.

### 👔 Модуль вакансий
* **Раздел вакансий:** Интерактивная страница со списком актуальных вакансий, требованиями и условиями работы.
* **Форма отклика:** Соискатели могут быстро отправить заявку, после чего администрация получает уведомление на email.

### 🔒 Панель администратора
* **Управление записями:** Переключение статусов бронирования (`в ожидании` ➔ `подтверждено` ➔ `выполнено` ➔ `отменено`) с автоматической отправкой писем клиентам.
* **Управление мастерами (CRUD):** Возможность добавлять, редактировать и удалять барберов, менять их ранг и загружать фотографии (старые файлы автоматически удаляются с сервера при замене).
* **Управление услугами и сортировка:** Редактирование категорий и цен, а также интерактивная сортировка услуг на сайте методом Drag-and-Drop (перетаскиванием).
* **Многостраничный экспорт в Excel:** Выгрузка детальной аналитики с помощью пакета **Laravel-Excel** на 5 разных листов:
  1. **Список бронирований:** Полная история записей со статусами и деталями.
  2. **Статистика мастеров:** Объемы работы и выручка по каждому барберу.
  3. **Статистика услуг:** Популярность конкретных стрижек и процедур.
  4. **Статистика клиентов:** Частота визитов, сгруппированная по номерам телефонов.
  5. **Аналитика:** Общие бизнес-метрики

├── migrations/                  # Миграции базы данных с индексами для быстрого поиска
└── seeders/                     # Наполнение БД тестовыми данными (мастера, услуги)
resources/
├── css/                         # Стили Tailwind и кастомные стили для капчи
├── js/                          # Скрипты (анимация карусели отзывов, калькулятор времени)
└── views/                       # Шаблоны Blade, разметка страниц и компонентов

Системные требования

Убедись, что у тебя установлены PHP 8.2+, Composer и Node.js (v18+).
1. Клонирование репозитория

git clone [https://github.com/твой_логин/gidra-barbershop.git](https://github.com/твой_логин/gidra-barbershop.git)
cd gidra-barbershop

2. Установка зависимостей

# Установка PHP-пакетов
composer install

# Установка JS-пакетов
npm install

3. Настройка окружения

Создай файл конфигурации из шаблона:

cp .env.example .env

Открой .env и укажи настройки базы данных, почты и ключи капчи:
Фрагмент кода

DB_CONNECTION=sqlite
# или настройки MySQL

NOCAPTCHA_SITEKEY=твой_публичный_ключ_recaptcha
NOCAPTCHA_SECRET=твой_секретный_ключ_recaptcha

MAIL_MAILER=smtp
MAIL_HOST=smtp.mailtrap.io
# ...остальные настройки почты

4. Настройка базы данных и наполнение демо-данными

Сгенерируй ключ приложения, запусти миграции и сидеры (они создадут дефолтных мастеров и услуги):

php artisan key:generate
php artisan migrate --seed

5. Создание символической ссылки для картинок

Чтобы фотографии мастеров отображались на сайте, выполни:

php artisan storage:link

6. Запуск фронтенда и сервера

Запусти сборщик стилей/скриптов и локальный сервер Laravel:

# В первом терминале (сборка фронтенда)
npm run dev

# Во втором терминале (запуск сервера)
php artisan serve

Вход в панель управления осуществляется через подвал.

логин: admin@admin.com

пароль: password123

Открой в браузере: http://127.0.0.1:8000.
